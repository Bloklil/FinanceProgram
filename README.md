# 💰 Finance Program

Простое веб-приложение для учёта личных финансов.

Программа позволяет:

* управлять счетами
* добавлять категории
* учитывать доходы и расходы
* просматривать баланс
* фильтровать транзакции

---

# 🚀 Технологии

Frontend

* React
* Vite

Backend

* Java
* Spring Boot

База данных

* H2 / PostgreSQL (в зависимости от конфигурации)

# ⚙️ Запуск проекта

## 1. Запуск через IDE

Открой проект в **IntelliJ IDEA** и запусти класс:

```
FinanceProgramApplication
```

После запуска открой браузер:

```
http://localhost:8080
```

---

# ▶️ Запуск через JAR файл

Можно запускать приложение **без IDE**.

### 1. Перейти в папку с jar (укажи свой путь до папки)

```
cd "C:\...\FinanceProgram\target"  
```

### 2. Запустить приложение

```
java -jar FinanceProgram-0.0.1-SNAPSHOT.jar
```

После запуска открыть в браузере:

```
http://localhost:8080
```

---

# 💡 Удобный запуск (Windows)

Можно создать файл **run.bat** рядом с jar.

```
java -jar FinanceProgram-0.0.1-SNAPSHOT.jar
pause
```

Теперь приложение можно запускать **двойным кликом**.
